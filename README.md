# AI Spend Audit

AI Spend Audit is a modern SaaS-style web application that helps startups analyze and optimize their AI subscription spending. The platform identifies overspending opportunities, estimates potential savings, and recommends more cost-effective AI tooling strategies.

## Features

- AI spend analysis dashboard
- Dynamic savings calculations
- Personalized optimization recommendations
- Modern responsive UI
- Shareable audit reports
- Lead generation workflow for Credex

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- React
